# 一、项目目标（Objective）

本项目拟构建一个交互式网页系统，用于**可视化电磁场在介质边界处的行为**，重点展示以下核心物理关系：

- 势函数 φ 的连续性
- 电场 (\mathbf{E} = -\nabla \varphi) 的空间变化
- 电位移 (\mathbf{D} = \varepsilon \mathbf{E}) 的介质响应
- 边界条件：
  - 切向电场连续
  - 法向电位移连续（无自由电荷时）
  - 法向电场发生跳变

系统应从“空间分布 + 函数剖面 + 数值验证”三个层面，构建统一认知。

------

# 二、核心物理场景（唯一指定）

## 👉 点电荷 + 平面介质分界面

### 场景定义

- 空间分为两部分：
  - 上半空间：介电常数 (\varepsilon_1)
  - 下半空间：介电常数 (\varepsilon_2)
- 分界面：
  [
  z = 0
  ]
- 在上半空间放置一点电荷 (q)

------

### 选择该场景的理由

该模型具备以下优势：

1. **结构简单但物理现象完整**
   - 单一源 + 单一界面
2. **可同时展示所有关键边界效应**
   - 场线折射
   - 势函数连续
   - 法向分量跳变
3. **对称性适中**
   - 保留球对称（点电荷）
   - 被平面界面打破 → 产生可观察畸变
4. **适用于3D可视化与剖面分析**

# 四、剖面系统设计（核心创新点）

## 1. 剖面定义方式

### 👉 唯一推荐方案：法向剖面

定义一条穿过界面的直线：

[
x = x_0,\quad y = y_0,\quad z \in [-L, L]
]

即：

> 一条垂直于界面 (z=0) 的直线

------

## 2. 剖面生成逻辑

- 用户在3D视图中选择或拖动一个点
- 系统自动生成：
  - 通过该点的法向直线
- 沿该直线采样场量

------

## 3. 剖面图显示内容（右下区域）

### 必须显示三条曲线：

------

### （1）势函数 φ(z)

- 连续
- 平滑

------

### （2）法向电场 (E_n(z))

- 在界面处发生跳变

------

### （3）法向电位移 (D_n(z))

- 连续（无自由电荷条件下）

------

## 4. 剖面系统的物理意义

该模块实现：

> 将三维电磁场映射为一维函数，从而显式验证边界条件

# 五、边界条件可视化目标

系统应能够清晰表达以下关系：

------

## 1. 势函数连续性

[
\varphi_1 = \varphi_2
]

------

## 2. 电场切向连续

[
E_{1t} = E_{2t}
]

------

## 3. 电位移法向连续（无自由电荷）

[
D_{1n} = D_{2n}
]

------

## 4. 电场法向不连续

[
E_{1n} \ne E_{2n}
]

## 4.4 剖面采样算法

```js
function sampleSliceLine(sliceLine, params) {
  const data = [];

  for (let z = zMin; z <= zMax; z += dz) {
    const { phi, E, D } = computeField(x0, y0, z);

    const En = E.z;
    const Dn = D.z;

    data.push({ z, phi, En, Dn });
  }

  return data;
}
```
